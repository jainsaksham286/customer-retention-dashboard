# Customer Retention and Churn Analysis Dashboard

## 📊 Project Overview
This project analyzes telecom customer churn using Power BI.

## 📁 Repository Structure
- Dashboard → Contains Power BI (.pbix) file
- Report → Contains final project report
- Dataset → Contains Telco Customer Churn dataset

## 🛠 Tools Used
- Microsoft Power BI
- DAX
- Power Query

## 🎯 Key Features
- Churn Rate & Retention Rate Analysis
- Contract Type Segmentation
- Tenure-Based Analysis
- Payment Method Comparison
